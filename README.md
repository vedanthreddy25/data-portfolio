## Getting Started

**Prerequisites:** Node.js and npm installed

**Installation:**

```bash
git clone https://github.com/vedanthreddy25/data-portfolio.git
cd data-portfolio
npm install
```

**Run Locally:**

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

**Build for Production:**

```bash
npm run build
```

**Deploy to GitHub Pages:**

```bash
npm run deploy
```
